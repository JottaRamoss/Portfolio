# 🚀 Portfólio — João Gabriel Salustiano Ramos

> Site de portfólio pessoal desenvolvido do zero para apresentar projetos, habilidades técnicas e facilitar o contato com recrutadores e clientes. Design moderno, minimalista e focado em experiência do desenvolvedor.

## 👨‍💻 Sobre

Portfólio pessoal de **João Gabriel Salustiano Ramos**, desenvolvedor Front-End com formação técnica em Desenvolvimento de Sistemas. O site foi construído com HTML, CSS e JavaScript puro — sem frameworks — como demonstração prática das habilidades técnicas apresentadas nele.

---

## 🗂️ Seções

| Seção | Descrição |
|---|---|
| 🏠 **Hero** | Apresentação com card de código animado e status de disponibilidade |
| 👤 **Sobre** | Resumo profissional, formação e card de perfil |
| ⚙️ **Skills** | Stack completo dividido em Front-End, Back-End, Banco de Dados e Ferramentas |
| 💼 **Projetos** | Projetos em destaque com filtro "Clientes Reais" |
| ✨ **Diferenciais** | Código limpo, performance, responsividade, visão Full-Stack |
| 📬 **Contato** | Formulário + links para E-mail, LinkedIn e GitHub |

## ✨ Destaques do projeto

- **Card de código animado** no hero simulando um arquivo TypeScript real
- **Filtro de projetos** por categoria (Todos / Clientes Reais)
- **Contadores animados** — Projetos, Site Vendido, Tecnologias
- **Design minimalista** com tema escuro e detalhes em verde neon
- **100% responsivo** com abordagem mobile-first
- **Formulário de contato** funcional
- **Zero frameworks** — construído apenas com HTML, CSS e JS puro
